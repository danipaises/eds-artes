# Quiz Escolar

Aplicação React/Vite com fase de grupos em todos contra todos.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Produção

```bash
npm run build
```

O Netlify usa `npm run build` e publica a pasta `dist`, conforme `netlify.toml`.

## Teste local sem Firebase

Abra o site com `?sync=local` para testar várias abas no mesmo navegador usando armazenamento local. O modo normal usa Firebase/Firestore.
